# workstation-setup
Scripts to configure a linux workstation for devops/sre/sysadmin

First install git
```bash
sudo apt install git
```

Use:

```bash
# Clone repo
git clone https://github.com/elniko77/workstation-setup-linux.git

# Run cli installation (debian,ubuntu)
./run_cli.sh

# Run gui installation for gnome (flatpaks, gnome settings, etc)
./run_gui.sh
```
